Instalación Software Portable.

Vamos a instalar dos softwares portables, ambos los comprobaremos en una maquina virtual.ㅤ

1-Instalación de Gparted de manera portable:

Gparted es un software que nos permite crear, editar, eliminar particiones… Se puede utilizar instalandolo en el equipo o utilizandolo de forma portable como vamos a ver.

Para utilizarlo de forma portable, nos iremos a su pagina de descarga Aquí y descargaremos la versión que se ajuste a nuestro equipo (32 o 64bits)

ㅤ

Ahora utilizaremos un software para bootear un USB como puede ser Rufus, Unetbooting , incluiremos dentro el archivo .iso y ya estaría listo para su uso. Por ultimo iniciamos el equipo con Gparted. En esta ocasión esta practica no cubrira este paso, aunque dejare una captura de como configurar Unetbooting. Vamos a instalarlo en una maquina virtual para ver su interfaz. Para ello simplemente agregamos el disco a la maquina y lo iniciamos.

3-Comparativa entre softwares:

Gparted es un software tanto de instalación como portable, eso es una gran ventaja ya que si nos gusta el programa podriamos instalarlo en el equipo.

En cambio Clonezilla solo se utiliza de forma portable.

Ambos softwares tienen en comun que ocupan muy poco espacio y no requieren de apenas requisitos.
